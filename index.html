Workspace 主工作台

Browse, select, and launch comparison from one place. 在同一个工作台里浏览、筛选并拉起对比。

The main browser combines directory tree, quick access, preview, search, and multi-column browsing. From the browser area you can select files across columns and open a dedicated compare window without breaking the workflow. 主界面把目录树、快速访问、预览、搜索和多栏浏览整合在一起。你可以直接在浏览区跨栏选图， 然后拉起独立对比窗口，不必在多个工具之间来回切换。

  • Up to 4 browser columns for large local image sets, with natural sort and path jump. 最多 4 栏浏览大型本地图像集，支持自然排序与路径直达。
  • Cross-column selection lets you pick any target pair or group before entering compare mode. 跨栏多选让你可以自由挑出任意两张或多张目标图，再进入对比模式。
  • Local-first design keeps previews, caches, and metadata on your Mac. 本地优先设计把预览、缓存与元数据处理都保留在你的 Mac 上。
Compare 多图对比

Choose the right layout for every review pass. 按场景切换最合适的对比布局。

Use regular side-by-side comparison for day-to-day review, then switch to diff view, ROI, pixel readout, and multi-image layouts when the job needs deeper inspection. 日常复核可以直接使用普通两图并排模式；需要更深入检查时，再切换差异图、ROI、像素读数和多图布局。

Standard 2-image mode 普通两图模式

Add the first image, then the second one to enter a clean side-by-side layout with synchronized zoom and pan. It is built for direct version checks, before-and-after review, and fast visual comparison without extra clutter. 添加第一张图后，再放入第二张图，就能进入清晰的并排模式，支持同步缩放与同步平移， 适合修图前后对照、版本复核和日常快速比较。

Sync zoom 同步缩放 Sync pan 同步平移 Side by side review 并排复核

Difference view 两图差异

Dedicated tolerance diff mode helps reveal subtle edits, version mismatches, and local rendering changes faster than visual guessing alone. Tune the threshold to suppress harmless noise and keep the focus on pixels that actually changed. 专门的容差差异模式可以更快锁定细微改动、版本偏差和局部渲染变化，不用只靠肉眼来回猜测。 你还可以调节容差，减少无关噪声，把注意力放在真正发生变化的像素区域。

Tolerance control 容差调节 Version review 版本校对

Three-up review 三图对比

Keep three aligned versions on one screen when two candidates are not enough to make the call. It works well for source, edit, and export review, or for checking several render passes at the same stage. 当两张图还不足以判断时，三图同屏可以把更多版本直接放到同一视野里。它很适合原图、编辑稿、 导出稿联看，也适合在同一阶段检查多个渲染结果。

Four-up 2×2 layout 四图 2×2 对比

A balanced layout for approvals, side-by-side variants, and team review sessions. Each panel stays readable on a notebook screen, so it is practical for A/B/C/D checks without opening extra windows. 平衡的 2×2 排布适合多版本审稿、A/B/C/D 方案筛选和团队评审。每个画面在笔记本屏幕上仍然保持可读， 方便不额外开窗就完成多版本比较。

Five-up screening 五图筛选

Show more candidates in one pass for culling, style checks, and generated image screening. It is useful when you want to narrow a burst or AI-generated set quickly before moving finalists into deeper inspection. 单次放下更多候选图，适合选片、风格筛查和生成结果初筛。你可以先快速缩小候选范围，再把最终入围图 带进更深入的像素检查或差异对比。

Six-image compare wall 六图同屏对比

Maximize screen space for high-throughput comparison of large, high-resolution image sets. This layout is ideal for final consistency sweeps across big batches where speed matters as much as detail. 把全屏空间利用到最大，适合高频横向比较大量高分辨率图片。对于需要快速扫完整批素材、 同时又要保持细节判断的场景，这个布局尤其高效。

Inspect and Refine 分析与处理

Move from viewing to actionable image work. 从看图直接进入可执行的图像处理。

The app does more than display files. High-zoom pixel values, batch cleanup, crop, and resize tools stay close to the comparison workflow. 软件不只是展示文件。高倍率像素值显示、批量整理、裁剪和尺寸调整都紧贴在对比工作流旁边， 减少工具切换成本。

High-zoom pixel values 高倍率显示像素值

Once a comparison pair is open, zoom deep into either side and keep RGB, HSV, luminance, histogram, ROI statistics, and 9×9 pixel data visible at the same time. It is designed for users who need exact pixel readout instead of only approximate visual judgment. 打开对比图组后，继续放大任意一侧时，RGB、HSV、亮度、直方图、ROI 统计和 9×9 像素数据会同时保留在界面上， 方便一路从整图看到具体像素值，而不是只做大致目测。

RGB / HSV / Y ROI Pixel grid values 像素网格值

Batch rename 批量重命名

Preview insert, delete, replace, clear, and sequence rules before applying changes to your file names. This keeps large folders consistent while reducing the risk of destructive rename mistakes. 插入、删除、替换、清除和序列化规则都可以先预览，再统一执行文件名修改。 对需要批量整理的大型文件夹来说，这种方式更稳妥，也更不容易误改名称。

Preview first 先预览 Rule based 规则驱动

Crop tool 图片裁剪

Freeform crop with live preview, aspect presets, and export choices including new file, clipboard, or in-place overwrite. It works well for quick cleanup as well as repeated output prep with fixed aspect targets. 支持自由裁剪、实时预览、宽高比预设，以及导出到新文件、剪贴板或覆盖原图等处理方式。 无论是临时修边，还是反复输出固定比例素材，都能更顺手地完成。

Live preview 实时预览 Aspect presets 比例预设

Resize images 图片大小调整

Resize by exact pixels or percentage, keep aspect ratio, and choose the scaling algorithm that matches the output requirement. It is practical for web delivery, thumbnails, dataset prep, and export specs. 支持按精确像素或百分比调整尺寸，可保持宽高比，并选择匹配输出需求的缩放算法。 对网页上传、缩略图生成、数据集准备和交付规格输出都很实用。

More Highlights 更多亮点

Built for real-world image libraries. 为真实图像库工作流而设计。

Beyond comparison and editing tools, Domino Image Viewer is designed to open more files, reduce context switching, and keep review workflows fast on Mac. 除了对比和处理工具外，Domino Image Viewer 也强调更广的打开能力、更少的切换成本， 以及更顺畅的 Mac 本地工作流。

100+ image formats 支持超过 100 种图片格式

Open a broad range of image files through macOS-native decoders plus extra support for pro and special formats such as RAW, PSD, WebP, AVIF, SVG, AI, ICO, TIFF, and more. This makes mixed-format folders easier to inspect without splitting them into separate tools first. 通过 macOS 原生解码能力与额外专业格式支持，软件可以打开大量常见与进阶图片文件， 包括 RAW、PSD、WebP、AVIF、SVG、AI、ICO、TIFF 等多种类型。面对混合格式文件夹时， 你也不需要先拆分到不同工具里再处理。

Browse images inside archives 支持压缩包图片文件直接查看

Supported archives can be opened directly so you can preview and compare image assets without extracting the whole package first. It is especially useful for downloaded asset bundles, comics, or packaged review sets. 支持的压缩包可以直接打开浏览，让你无需先完整解压，就能预览和对比其中的图片资源。 对下载素材包、漫画包或打包后的审稿资源来说，这样的流程会更直接。
